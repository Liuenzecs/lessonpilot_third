# Word 导出规格

## 目标

导出文件必须让老师可以提交、打印或继续小幅修改。

当前 MVP 以 `.docx` 为唯一正式导出格式。

## 数据来源

导出读取：

- `Document.content`
- `Task` 元信息

导出服务：

- `apps/api/app/services/export_service.py`

## 教案导出

包含：

- 标题
- 学科、年级、课时、课型
- 教学目标
- 教学重难点
- 教学准备
- 教学过程
- 板书设计
- 教学反思

规则：

- 教学过程使用表格。
- 家教场景可简化列。
- 教学反思即使为空，也保留填写区域。
- 默认只导出已确认 section，反思空白区除外。

## 学案导出

包含：

- 标题
- 学科、年级、班级、姓名、日期
- 学习目标
- 重点难点预测
- 知识链接
- 学习流程
- 达标测评
- 拓展延伸
- 自主反思

规则：

- 班级、姓名、日期可留空但要有位置。
- 学习流程中的自主学习、合作探究、展示提升来自 `learning_process`。

## 未来模板导入方向

学校模板应被解析成：

- 字段映射。
- section 顺序。
- 表格布局。
- 必须保留的空白区域。
- 静态学校文案。

不要把学校模板硬编码成全局唯一格式。
