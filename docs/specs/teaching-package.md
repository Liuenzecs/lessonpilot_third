# 上课包规格

## 目标

从一份已确认教案继续生成可上课材料，让老师不用重新复制粘贴：

- 学案草稿
- PPT 大纲/课件文案
- 课堂口播稿/提问串

## 生成规则

- 输入为当前 `LessonPlanContent` 和 `Task` 元信息。
- 教案核心内容不足或关键 section 未确认时，返回老师可理解的提示。
- 生成结果默认 `pending`，由老师确认、编辑或复制。
- PPT v1 只生成结构化大纲，不生成真实 `.pptx`。

## 数据形态

上课包以结构化 JSON 保存，包含：

- `study_guide`：可直接进入现有学案编辑/导出链路
- `ppt_outline`：slide 标题、要点、课堂活动、备注
- `talk_script`：导入语、问题链、过渡语、小结语
