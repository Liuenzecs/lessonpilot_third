# LessonPilot 隐私政策

> 最后更新：2026-05-05

## 我们收集的信息

### 你主动提供的信息

- **账户信息**：姓名、邮箱地址
- **备课内容**：你创建的教案、学案、教学日历等
- **上传资料**：你上传的个人备课资料
- **反馈信息**：你提交的使用反馈和心情评价
- **风格偏好**：你设置的教学风格偏好

### 自动收集的信息

- **使用统计**：生成次数、导出次数、页面访问记录
- **技术信息**：浏览器类型、设备类型、访问时间

## 我们如何使用信息

- 提供和改进 AI 备课服务
- 发送账户相关通知（验证邮件等）
- 统计服务使用情况，优化成本控制
- 收集反馈以改进产品质量

## 我们不会做的事

- **不会**将你的备课内容用于 AI 模型训练
- **不会**向第三方出售或分享你的个人数据
- **不会**在你的备课内容中插入广告
- **不会**未经你同意公开你的备课内容

## 数据存储与安全

- 数据存储在中国大陆服务器上
- 密码使用 bcrypt 加密存储
- API 通信使用 HTTPS 加密
- 我们采取合理的技术措施保护你的数据安全

## 数据删除

你可以随时在设置页的"数据管理"中导出或删除你的全部数据。删除操作不可恢复。

## 联系我们

如对隐私政策有任何疑问，请发送邮件至 hello@lessonpilot.com。
