# LessonPilot 成本预算方案

> 最后更新：2026-05-05

## 成本来源

LessonPilot 的运营成本主要来自 AI 模型 API 调用：

| 项目 | 提供商 | 模型 | 估算费率 |
|------|--------|------|---------|
| 文本生成 | DeepSeek | deepseek-v4-flash | ~¥0.001/1K prompt tokens + ¥0.002/1K completion tokens |
| 文本生成 | MiniMax | MiniMax-Text-01 | ~¥0.0015/1K prompt tokens + ¥0.003/1K completion tokens |
| Embedding | 本地 BGE | bge-m3 | 免费（CPU 推理） |

## 单次生成成本估算

一次完整的教案生成（约 6 个 section）：
- 平均 token 消耗：~15,000 tokens（prompt + completion）
- 估算成本：~¥0.02-0.05

一次学案生成（约 9 个 section）：
- 平均 token 消耗：~20,000 tokens
- 估算成本：~¥0.03-0.08

## 月度预算

- **当前月度预算上限**：¥500/月
- **告警阈值**：80%（¥400），触发日志告警
- **限流阈值**：95%（¥475），自动降低新生成速率

## 按用户规模估算

| 活跃老师数 | 每人每月 30 次生成 | 月成本估算 |
|-----------|-------------------|-----------|
| 10 | 300 次 | ~¥10-25 |
| 50 | 1,500 次 | ~¥50-125 |
| 100 | 3,000 次 | ~¥100-250 |
| 500 | 15,000 次 | ~¥500-1,250 |

## 成本控制措施

1. **每日配额**：每位老师每天最多 20 次生成
2. **月度配额**：每位老师每月最多 300 次生成
3. **月度总预算**：超出预算 95% 自动限流
4. **本地 Embedding**：使用本地 BGE 模型，零 API 成本
5. **成本追踪**：每次 LLM 调用记录 token 数和成本

## 可持续性

当前以个人资金支持运营。如果用户规模增长导致成本超出个人承受能力，我们将：
1. 优先优化 prompt 减少 token 消耗
2. 评估开源/本地模型替代方案
3. 考虑接受公益基金或捐赠支持

**不会做的事**：向老师收费、插入广告、出售数据。
