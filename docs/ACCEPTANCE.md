# 手动验收脚本

## 当前主链路验收

目标：确认 LessonPilot 从创建到导出已经稳定可演示。

### 1. 登录与备课台

- 注册或登录一个测试账号。
- 进入备课台。
- 确认空状态、任务卡片、搜索、复制、删除入口显示正常。

通过标准：
- 页面无异常跳转。
- 未登录时会回到登录页。
- 已登录时能看到备课台。

### 2. 创建备课

- 点击“开始备课”。
- 选择语文、年级、课题。
- 分别测试教案、学案、教案+学案。
- 填写补充说明。

通过标准：
- 能创建任务并进入编辑器。
- 新任务会自动创建对应文档。

### 3. Section 级生成

- 新建任务后等待自动生成。
- 观察每一节生成过程。

通过标准：
- 事件按 `section_start / section_delta / section_document / section_done / document_done` 进行。
- 每个 section 完成后内容持续可见。
- 不出现“生成后短暂消失”。

### 4. 编辑与保存

- 修改一个文本 section。
- 修改一个列表 section。
- 修改教学过程或题目 section。
- 等待自动保存。
- 刷新页面。

通过标准：
- 修改不会丢失。
- 保存状态能回到已保存。
- 学案 `self_study / collaboration / presentation` 均保存到 `learning_process`。

### 5. 局部改写

- 对一个已有 section 点击调整。
- 测试改写、补充展开、压缩表达。

通过标准：
- 改写前会保存本地改动。
- 改写完成后 section 内容更新。
- 文档版本增加。

### 6. RAG 引用

- 使用已入库知识能命中的课题，例如《红楼梦》或人物分析。
- 生成后查看 section 顶部参考资料。

通过标准：
- 命中 RAG 时 `section_references` 非空。
- 前端显示参考资料徽标和 tooltip。
- `RAG_ENABLED=false` 时能优雅退化为普通生成。

### 7. Word 导出

- 确认若干 section。
- 点击导出 Word。
- 打开 Word 文件。

通过标准：
- 文件能打开。
- 标题、元信息、教学目标、过程表格等格式正常。
- 仅导出已确认内容，反思保留空白区。

## 验收记录规则

验收通过后再更新 `docs/NEXT.md` 为“等待下一步指示”或下一个经用户确认的 Cycle。

不要在未验收前自动进入模板库运营化、知识包扩学科或部署阶段。
