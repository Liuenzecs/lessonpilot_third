# 老师真实工作流

## 公立校正式备课

1. 接到课题或备课任务。
2. 找教材、教参、往年教案。
3. 按学校模板写教案。
4. 调整教学目标、重难点和过程。
5. 导出或整理 Word。
6. 提交给备课组、教研组或学校系统。

LessonPilot 切入点：
- 创建备课时生成结构化骨架。
- RAG 引用教研资料。
- 逐节确认内容。
- 按学校模板导出 Word。
- 导出前检查必填项。

## 家教备课

1. 确认学生年级和学习问题。
2. 确定本次课主题。
3. 设计讲解顺序。
4. 准备练习题。
5. 上课时边讲边改。

LessonPilot 切入点：
- 生成讲解路径。
- 生成学生讲义和练习。
- 简化正式教案语言。
- 支持按学生基础调整难度。

## 机构老师备课

1. 按课程安排准备多节课。
2. 改编旧课件或讲义。
3. 生成课堂练习和课后作业。
4. 多班复用并微调。

LessonPilot 切入点：
- 复制旧备课。
- 复用常用 section。
- 一份备课生成教案、学案、PPT 大纲。
- 记录已上过和效果好版本。

## 高频操作清单

老师最常做的不是长篇聊天，而是：

- 改成更正式。
- 改成更适合学生。
- 加课堂提问。
- 加学生可能回答。
- 补设计意图。
- 调整到 40/45 分钟。
- 把题目分层。
- 按学校格式导出。

这些操作应优先变成 LessonPilot 的显性能力。
