# Phase 7-10：学校格式与个人备课资产闭环

## Summary

本阶段把学校格式记忆、教学质量检查 2.0、上课包生成、个人资料库合并为一个连续大阶段。目标是让老师把自己的学校模板和旧资料带进 LessonPilot，并从一份教案继续得到可提交、可上课、可复用的一组材料。

## Key Outcomes

- 学校模板能被上传、解析、保存，并用于 Word 导出。
- 导出前体检升级为目标、过程、评价一致性检查。
- 已确认教案能生成学案、PPT 大纲、课堂口播稿。
- 个人旧 `.docx` / `.pptx` 资料能预览、确认入库，并作为私有资产复用。
- 所有导入或生成的新增内容默认待确认，老师保留控制权。

## Scope

本阶段只做语文 MVP 内的老师个人备课资产闭环。学校模板 v1 做结构化导出规格，不做像素级 Word 复刻；PPT v1 输出大纲和课件文案，不生成真实 `.pptx`；个人资料库 v1 支持 `.docx` 和 `.pptx`，PDF/OCR 后置。

## Acceptance

- 上传学校模板后能看到字段映射、栏目顺序、教学过程表格列和无法识别内容。
- 保存个人模板后，导出 Word 可按模板顺序组织内容，并保留必要空白区。
- 质量检查 2.0 能发现目标空泛、过程未承接目标、测评未覆盖目标、活动不可执行、重难点缺失等问题。
- 一份完整教案能生成三类上课包草稿，且默认 pending。
- 上传旧资料后，老师能看到预览、未识别内容、复用建议，并确认保存为个人资产。
- 个人资产与私有知识只对当前用户可见。
