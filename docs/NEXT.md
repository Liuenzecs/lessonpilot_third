# 当前任务：Phase 7-10 学校格式与个人备课资产闭环（待验收）

## 本阶段目标

让 LessonPilot 从“能生成和导出”升级为“记得老师的学校格式和个人资料，并能产出可提交、可上课的一套材料”：

- 学校 Word 模板上传、识别、保存与模板化导出
- 教学质量检查 2.0：目标、过程、评价一致性
- 从教案生成上课包：学案、PPT 大纲、课堂口播稿
- 个人资料库：旧 `.docx` / `.pptx` 上传、预览、私有复用

## 已授权范围

- 仍聚焦语文 MVP，不扩展全学科
- 学校模板 v1 做结构化模板化导出，不承诺像素级复刻原 Word
- PPT v1 只生成大纲/课件文案，不生成真实 `.pptx`
- 个人资料库 v1 支持 `.docx` 与 `.pptx`，PDF/OCR 留到后续
- 质量检查 2.0 优先规则化和结构化检查
- 阶段结束停在验收口，不自动提交 / 推送，不进入下一阶段

## 已实现

- 备课台新增“学校模板”和“个人资料库”入口。
- 学校模板支持 `.docx` 预览、保存为个人模板、列表和删除。
- Word 导出支持 `template_id`，可按个人学校模板渲染元信息、栏目顺序、教学过程表格列和空白区域。
- 导出前体检返回 `alignment_map`，能提示目标、过程、评价一致性问题。
- 教案编辑器新增学校模板选择和上课包面板。
- 上课包支持从已确认教案生成学案草稿、PPT 大纲、课堂口播稿。
- 个人资料库支持 `.docx` / `.pptx` 上传预览、确认保存、列表和删除，资料只对当前用户可见。

## 验收标准

- 老师上传学校 `.docx` 模板后，能看到结构化预览并保存为个人模板
- 使用个人模板导出的 Word 能保持栏目顺序、元信息区、教学过程表格列和必要留空区域
- 导出前体检能指出至少 5 类教学一致性问题，并给出老师能直接照做的修复建议
- 一份完整教案能生成学案、PPT 大纲、课堂口播稿三类上课包草稿，且默认 pending、可编辑
- 老师上传旧 `.docx` 或 `.pptx` 资料后，能看到预览、未识别内容和可复用建议
- 个人资料和私有知识只对当前用户生效，不污染公共知识包或其他用户数据
- 现有创建、生成、旧教案导入、RAG 引用、编辑保存、默认 Word 导出链路不退化
- 前后端测试、类型检查、构建与静态检查通过

## 暂不进入下一阶段

Phase 7-10 完成后停在用户验收清单，不自动进入全学科、真实 PPT 导出、PDF/OCR、学校后台或管理系统阶段。
