# 教案/学案质量标准

## 总原则

一份好初稿应让老师进入“修改”状态，而不是“重写”状态。

评价质量时看四点：

1. 是否贴合课题。
2. 是否能在真实课堂执行。
3. 是否符合老师的使用场景。
4. 是否能减少导出和提交前返工。

## 教案质量标准

### 教学目标

合格：
- 2-3 条。
- 可观察、可评价。
- 与课题具体内容相关。

不合格：
- “提高学生核心素养”这类空话。
- 三条目标只是换句式重复。
- 与教学过程没有对应关系。

### 教学重难点

合格：
- 教学重点指向本课核心内容。
- 教学难点解释学生为什么会难。

不合格：
- 重点和难点完全一样。
- 只写“理解课文内容”。

### 教学过程

合格：
- 至少 3 个环节。
- 每个环节有教师活动、学生活动、设计意图、时间。
- 环节之间有推进关系。
- 总时长接近课时。

不合格：
- 学生活动始终是“认真听讲”。
- 没有真实问题或任务。
- 时间安排明显不合理。

### 板书设计

合格：
- 能直接写到黑板上。
- 呈现关键词、结构或阅读路径。

不合格：
- 只复制标题。
- 过长，不像板书。

## 学案质量标准

### 学习目标

合格：
- 使用“我能...”口吻。
- 学生能读懂。
- 对应课堂任务。

### 学习流程

合格：
- 自主学习适合独立完成。
- 合作探究有讨论价值。
- 展示提升促进表达或迁移。

### 达标测评

合格：
- 检查核心目标。
- 题目、答案、解析基本完整。

### 拓展延伸

合格：
- D 级或选做。
- 不强制所有学生完成。

## 严重问题等级

- `P0`：错误课题、错误学科、结构损坏、内容不可用。
- `P1`：老师大概率需要重写，或导出后不能交。
- `P2`：可用但质量弱，需要补细节。
- `P3`：文案或格式优化。
